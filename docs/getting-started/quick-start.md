# Quick Start Guide

Get up and running with Runopt in just a few minutes. This guide will walk you through your first property analysis.

## Step 1: Access the Platform

1. Open your web browser
2. Navigate to [https://core.runopt.co](https://core.runopt.co)
3. Sign in to your account (or create one if you haven't already)

## Step 2: Start a New Analysis

### Option A: Property Search
1. Click "New Analysis" from the dashboard
2. Enter a property address in the search bar
3. Select the property from the dropdown results
4. Click "Analyze Property"

### Option B: CSV Upload
1. Click "Analyze Site CSV" button
2. Upload your CSV file with property data
3. Review the data preview
4. Click "Start Analysis"

## Step 3: Review Zoning Information

Once your property is loaded:

1. **View the Zoning Map**
   - Interactive map showing zoning districts
   - Color-coded zoning classifications
   - Property boundaries and lot lines

2. **Check Zoning Details**
   - Zoning district information
   - Building height restrictions
   - Floor area ratio (FAR) limits
   - Setback requirements

3. **Review Property Information**
   - Lot size and dimensions
   - Current building information
   - Tax assessment data

## Step 4: Use Civil Engineering Tools

### Storm Water Analysis
1. Click on "Storm Water" in the tools panel
2. Set your design parameters
3. View runoff calculations
4. Export storm water reports

### Utilities Planning
1. Select "Utilities" from the tools menu
2. Plan utility connections
3. Review infrastructure requirements
4. Generate utility layout plans

### Parking Analysis
1. Open the "Parking" tool
2. Enter parking requirements
3. Design parking layout
4. Calculate parking capacity

## Step 5: Get AI-Powered Insights

### Chat with Zonewise
1. Click the chat icon in the bottom right
2. Ask questions about your property:
   - "What are the zoning restrictions for this property?"
   - "Can I build a 5-story building here?"
   - "What are the parking requirements?"
3. Get instant, AI-powered answers

### Send Data to Chat
1. After analyzing a property, click "Send to Zonewise Chat"
2. Your property data will be shared with the chatbot
3. Continue the conversation with context

## Step 6: Export Your Results

### Generate Reports
1. Click "Export" in the top navigation
2. Select report type:
   - Zoning analysis report
   - Civil engineering summary
   - Complete project report
3. Download as PDF or CSV

### Save Your Project
1. Click "Save Project" to store your work
2. Add project notes and descriptions
3. Access your saved projects later

## Common Workflows

### Residential Development
1. Search for residential property
2. Check zoning for residential use
3. Analyze parking requirements
4. Plan storm water management
5. Export feasibility report

### Commercial Development
1. Find commercial property
2. Review commercial zoning
3. Analyze utility requirements
4. Plan parking and access
5. Generate development plan

### Mixed-Use Projects
1. Identify mixed-use zoning
2. Analyze different use types
3. Plan shared parking
4. Design utility systems
5. Create comprehensive report

## Tips for Success

### Before You Start
- Have the property address ready
- Know your project goals
- Gather any existing site data

### During Analysis
- Take notes on important findings
- Use the chat feature for questions
- Save your work frequently

### After Analysis
- Export all relevant reports
- Save your project
- Share results with your team
