# Installation & Setup

Runopt is a web-based platform that doesn't require traditional installation. This guide will help you get started with accessing and using Runopt.

## System Requirements

### Browser Requirements
- **Chrome**: Version 90 or higher
- **Firefox**: Version 88 or higher
- **Safari**: Version 14 or higher
- **Edge**: Version 90 or higher

### Recommended Specifications
- **RAM**: 8GB or more
- **Storage**: 1GB free space for caching
- **Internet**: Stable broadband connection
- **Display**: 1920x1080 or higher for optimal experience

## Accessing Runopt

### 1. Core Platform
The main Runopt platform is available at:
```
https://core.runopt.co
```

### 2. NYC Zoning App
For NYC-specific zoning analysis:
```
https://core.runopt.co/zoning-map
```

### 3. Landing Page
For general information and updates:
```
https://runopt.co
```

## Account Setup

### Creating an Account
1. Navigate to [https://core.runopt.co](https://core.runopt.co)
2. Click "Sign Up" in the top navigation
3. Fill in your details:
   - Email address
   - Password
   - Company name (optional)
   - Role/Title (optional)
4. Verify your email address
5. Complete your profile setup

### Authentication
Runopt uses secure authentication with:
- Email/password login
- JWT token-based sessions
- Automatic session refresh
- Secure password requirements

## First-Time Setup

### 1. Complete Your Profile
- Add your company information
- Set your preferred units (metric/imperial)
- Configure notification preferences

### 2. Explore the Dashboard
- Familiarize yourself with the main navigation
- Review available tools and features
- Check out the tutorial sections

### 3. Create Your First Project
- Click "New Project" from the dashboard
- Enter project details
- Start with a simple property search

## Browser Configuration

### Enable JavaScript
Ensure JavaScript is enabled in your browser for full functionality.

### Allow Location Services
For property search features, allow location access when prompted.

### Clear Cache (if needed)
If you experience issues, try clearing your browser cache:
- **Chrome**: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
- **Firefox**: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
- **Safari**: Cmd+Option+E

## Troubleshooting

### Common Issues

**Page not loading:**
- Check your internet connection
- Try refreshing the page
- Clear browser cache

**Login issues:**
- Verify your email and password
- Check if caps lock is on
- Try the "Forgot Password" option

**Slow performance:**
- Close unnecessary browser tabs
- Check your internet speed
- Try a different browser

### Getting Help
If you continue to experience issues:
- Contact support at support@runopt.co
- Check our documentation for help
- Visit our [GitHub repository](https://github.com/Runopt) for known issues
